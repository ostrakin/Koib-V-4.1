# VK Bot для КОИБ - Документация по новым модулям

## Обзор

Этот проект добавляет полноценный VK-чат-бот к существующей RAG-системе для документации КОИБ.

## Новые файлы

### 1. `setup.py` - Модуль инициализации для Google Colab

**Назначение:** Автоматическая настройка системы при запуске в Colab.

**Функции:**
- `mount_google_drive()` - Монтирует Google Drive
- `get_secrets()` - Читает секреты из Colab Secrets (GIGACHAT_CREDENTIALS, VK_GROUP_TOKEN)
- `get_system_prompt()` - Загружает system prompt из `/content/drive/MyDrive/Koib/prompt.txt`
- `check_faiss_index_exists()` - Проверяет наличие FAISS-индекса
- `initialize()` - Главная функция инициализации

**Использование:**
```python
from setup import initialize
engine, system_prompt = initialize()
```

---

### 2. `gigachat_client.py` - Клиент GigaChat API

**Назначение:** Работа с GigaChat через OAuth2.

**Классы:**
- `GigaChatClient` - Основной клиент с кэшированием токена

**Функции:**
- `call_gigachat(prompt, credentials)` - Удобная функция для вызова API

**Параметры запроса:**
- Temperature: 0.3
- Max tokens: 1024
- Timeout: 30 секунд

**Использование:**
```python
from gigachat_client import call_gigachat

answer = call_gigachat("Ваш вопрос", credentials)
```

---

### 3. `vk_bot.py` - VK Long Poll бот

**Назначение:** Полноценный чат-бот для ВКонтакте.

**Классы:**
- `UserSession` - Хранение сессии пользователя (model + history)
- `KoibVKBot` - Основной класс бота

**Функционал:**
- **Хранение сессий in-memory:** `{user_id: UserSession}`
- **История:** Последние 3 пары вопрос-ответ
- **Выбор модели:** 4 кнопки (КОИБ-2010, КОИБ-2017А, КОИБ-2017Б, Общий вопрос)
- **Изоляция моделей:** FAISS-поиск с фильтром по выбранной модели
- **Команды:** /start, "начать", "привет", "← Сменить модель"

**Использование:**
```python
from vk_bot import KoibVKBot

bot = KoibVKBot(engine, system_prompt)
bot.run()  # Запуск Long Poll
```

## Настройка в Google Colab

### Шаг 1: Colab Secrets

Добавьте в Colab Secrets (🔑 → "Add a new secret"):

| Название секрета | Значение |
|-----------------|----------|
| `GIGACHAT_CREDENTIALS` | `client_id:client_secret` в base64 |
| `VK_GROUP_TOKEN` | Токен группы ВКонтакте |

### Шаг 2: Подготовка Google Drive

Создайте структуру папок:
```
/content/drive/MyDrive/Koib/
├── docs/           # PDF/DOCX файлы документации
├── models/         # Кэш модели эмбеддингов (создаётся автоматически)
├── koib_rag_GLM1/  # FAISS-индекс и метаданные (создаётся автоматически)
└── prompt.txt      # System prompt (опционально)
```

### Шаг 3: Формат prompt.txt

```
Вы — ассистент для технической поддержки КОИБ.
Отвечайте ТОЛЬКО по документации.
---
Здравствуйте! Я виртуальный помощник по КОИБ.

Выберите модель из меню ниже.
```

Первый абзац до `---` используется как приветствие.

## Логика обработки сообщений

1. **Новый пользователь или /start:**
   - Сброс сессии
   - Отправка приветствия + 4 кнопки выбора модели

2. **Нажатие кнопки модели:**
   - Сохранение выбранной модели в сессию
   - Очистка истории
   - Ответ: "Вы выбрали [модель]. Задайте ваш вопрос."
   - Кнопка: "← Сменить модель"

3. **Текстовый вопрос:**
   - Если модель не выбрана → напомнить выбрать
   - Если `general` → GigaChat без RAG
   - Если конкретная модель → RAG поиск + GigaChat
   - Сохранение в историю (макс. 3 пары)

4. **Кнопка "← Сменить модель":**
   - Сброс сессии → возврат к шагу 1

## Изоляция моделей

**Критически важно:** Модели НЕ смешиваются.

```python
if model_key == "general":
    # Без RAG
    answer = call_gigachat(prompt, credentials)
else:
    # RAG с фильтром по модели
    llm_context = engine.ask_with_llm_context(
        question, 
        koib_model=model_key  # ← фильтр FAISS
    )
    prompt = f"""...
    Отвечай ТОЛЬКО по документации КОИБ-модели {model_key}.
    Если информации нет — скажи об этом.
    """
    answer = call_gigachat(prompt, credentials)
```

## Обработка ошибок

| Ситуация | Действие |
|----------|----------|
| GigaChat недоступен | Возврат "Сервис временно недоступен" |
| FAISS не нашёл результатов | "По данной модели информация не найдена" |
| Таймаут GigaChat (>30 сек) | "Сервис временно недоступен (таймаут)" |
| Пользователь пишет без выбора модели | Предложить выбрать модель |

## Примечания

- **Сессии хранятся только in-memory** - при перезапуске все сессии очищаются
- **Никаких персональных данных** не сохраняется
- **user_id ВКонтакте** используется только как ключ в словаре сессий
- **Модель эмбеддингов** кэшируется в `/content/drive/MyDrive/Koib/models/`
- **FAISS-индекс** хранится на Google Drive для быстрой загрузки
